# Week 3

This repo contains my Week 3 assignments and practice work.

## Topics Covered

MongoDB Basics – practiced CRUD operations like insert, update, delete, and find queries.

MongoDB Operators – worked with operators such as `$set`, `$unset`, `$push`, `$pull`, `$addToSet`, `$all`, and `$in`.

Nested Documents & Arrays – learned how to manage objects, arrays, and embedded fields inside MongoDB documents.

Express.js APIs – created REST APIs using Express.js for handling product-related operations.

Mongoose Models & Schemas – defined schemas, validations, timestamps, and connected MongoDB using Mongoose.

CRUD APIs – implemented Create, Read, Update, and Delete operations for products.

Error Handling – handled validation errors, cast errors, and server-side issues in Express applications.

API Testing – tested API routes using request files and HTTP methods like GET, POST, PUT, and DELETE.
